# Be *That* Guy

A Command Line or Terminal application to simulate productivity through numerous
technical-sounding processes that will impress your non-technical friends and
coworkers and give your nerd friends a chuckle. Your boss, however, might not
get the joke.

## Build Instructions

If you have g++ installed on your system, on either Windows or a Unix system,
simply clone the repo to your machine and run `make`.

    > git clone https://github.com/Deaboy/LookBusy
    > cd LookBusy
    > make

## Running

Just run the program in the Command Line or in the Terminal and press `ctrl+C`
to kill whenever you're done.
